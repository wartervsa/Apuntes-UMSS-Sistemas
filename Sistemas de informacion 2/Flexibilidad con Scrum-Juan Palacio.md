**Que es un proyecto?**
Se hace la diferenciacion entre operacion y proyecto.
Ambos dan como resultado un producto o servicio, los 2 tienen caracteristicas similares.- como estar realizados por personas, ejecutados con recursos limitados y se llevan a cabo siguiendo una estrategia de actuacion.
*Los productos o servicios realizados por las organizaciones pueden ser el resultado de operaciones o de proyectos.*(libro, pag.27)
- Las **operaciones** son productos o servicios que siguen el mismo plan de actuacion buscando un resultado con caracteristicas similares.
- A un **proyecto** se le agregan dos factores, que producen un resultado unico y tiene un intervalo de tiempo preestablecido.

Algunos ejemplos de proyectos:
- Diseno de un nuevo ordenador portatil.
- Construccion de un edificio 
- Diseno de un nuevo empaque para un producto.
	el libro define el exito de un proyecto asi.- *Se considera que un proyecto se ha desarrollado con éxito cuando se consigue la finalidad prevista, con el presupuesto y en las fechas que previamente se han estimado.*(libro, pag.31)
